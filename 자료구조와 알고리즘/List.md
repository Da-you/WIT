# List

- 선형적인 자료구조
- 순서가 존재
- array List, Linked List가 있음

## Array List

 - 배열기반의 리스트
   : 랜덤 엑세스 방식으로 리스트에 데이터가 몇개가 있든 인덱스만 알고있다면 항상 동일한 시간을 사용하여 데이터 접근 가능

  - 메모리 공간을 연속적으로 사용
    : 컴퓨터의 실제 물리적인 공간에서도 연속적인 공간을 사용해 연산이 쉬운 구조

  - 데이터의 삽입, 삭제시 O(n)의 시간이 소요
    : 데이터를 삽입, 삭제할 경우 데이터를 옮겨야 하기 때문에 시간이 오래걸림
